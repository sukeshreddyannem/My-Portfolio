Digital Clock using LPC2148 (On-chip RTC)

This project displays RTC information on an LCD, with functionality to edit time details by activating a switch designed using the interrupt EINT0. The LCD operates in 4-bit mode to reduce pin usage, ensuring efficient hardware interfacing. External peripherals include a 4x4 matrix keypad and switches, all implemented on an ARM development kit.

Tools Used: Keil uVision - C Compiler, Proteus, Flash Magic, LCD (HD44780), 4x4 Matrix Keypad, LPC2148 Microcontroller, AL Switch.

Train Ticket Booking using GCC Compiler

Ticket Booking System is a C-based console application designed to facilitate online train ticket reservations. This system provides users with essential functionalities such as sign-up, sign-in, booking tickets, canceling tickets, and checking train availability. The application ensures data persistence using file handling and employs single linked lists (SLL) for efficient management of train and passenger details.

Key Concepts: DMA, File Handling, Data Structures, Linked Lists, Multiple File Compilation.
